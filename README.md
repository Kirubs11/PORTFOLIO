## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Kirubs11/PORTFOLIO.git
    ```
2. Navigate to the project directory:
    ```bash
    cd PORTFOLIO
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
